Connect Claude Code
Using: ● Local

Claude Code registers MCP servers through the claude mcp add CLI. Run the command below in any terminal — it writes the server entry to your Claude Code config so the kiagent tools are available in your next session.

  1. 1 Open a terminal and run the command below. It registers KIAgent as an HTTP transport.
  2. 2 Restart any active Claude Code session (or open a new one).
  3. 3 Verify with /mcp — KIAgent should appear in the connected servers list.
Terminal command
claude mcp add kiagent --transport http --url http://127.0.0.1:7421/mcp --header "Authorization: Bearer ••••••••"
To register at the user (not project) scope, add --scope user. To remove later: claude mcp remove kiagent.
