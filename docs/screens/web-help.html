Getting started

kia gives you a stable HTTPS subdomain that points at the MCP server running on your machine. Sign in here with the same Google or Microsoft account you used in the kia desktop app, and your subdomain re-attaches automatically.

  1. Install kia desktop and sign in with Google or Microsoft.
  2. The desktop app provisions a subdomain (e.g. alice-a3f9c1.lokalkiagent.de) and opens a tunnel.
  3. Copy the HTTPS URL into any MCP-aware client — Claude.ai, Claude Code, VS Code.
Your subdomain belongs to the account, not the device. Reinstall kia anywhere; sign in with the same account; the hostname comes back.

MCP setup

Point any MCP client at your kia URL with the bearer token from your account page.

Claude Desktop

// ~/Library/Application Support/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "kia": {
      "transport": "sse",
      "url": "https://alice-a3f9c1.lokalkiagent.de/mcp",
      "headers": {
        "Authorization": "Bearer kia_pat_3f9...c1"
      }
    }
  }
}

Claude Code

# add the server with one CLI command
claude mcp add kia \
  --url "https://alice-a3f9c1.lokalkiagent.de/mcp" \
  --header "Authorization: Bearer kia_pat_3f9...c1"

DNS troubleshooting

If your account shows "Pending DNS" for more than a few minutes, propagation is slow somewhere upstream.

  • Most resolvers see the A record within 30 seconds. A small percentage need a few minutes.
  • Run dig +short alice-a3f9c1.lokalkiagent.de — you should get our broker IP.
  • Corporate DNS sometimes blocks dynamic subdomains; try from a phone hotspot.
  • If the tunnel says "down" but DNS resolves, the desktop app likely isn't running.

FAQ

I can't sign in

You may have cancelled the Google or Microsoft prompt — try again from the login page.

If you see "expired" or "interrupted", open the page fresh in the same browser you started in.

This site needs cookies. Check your browser allows cookies for lokalkiagent.de.

I deleted or reinstalled kia — what now?

Your web account survives, and your subdomain belongs to your account — not your device. Reinstall kia, sign in with the same provider, and your subdomain re-attaches automatically.

I want to release my subdomain

Go to your account page and use the Release subdomain button under the Subdomain card. The hostname returns to the available pool after a 30-day cooldown.

Can I bring my own domain?

Not yet. We're working on a CNAME flow for custom domains. For now everyone is on *.lokalkiagent.de.
