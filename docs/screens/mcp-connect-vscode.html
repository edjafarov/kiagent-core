Connect VS Code
Using: ● Local

VS Code's GitHub Copilot Chat reads MCP servers from a per-workspace JSON file. Drop the snippet below into .vscode/mcp.json and reload the window — KIAgent's tools will be available to Copilot Chat agents in that workspace.

  1. 1 In your workspace root, create .vscode/mcp.json (or open the existing one).
  2. 2 Add this server entry inside the servers object.
  3. 3 Reload the VS Code window: ⌘⇧P → Developer: Reload Window.
  4. 4 Open Copilot Chat in agent mode and confirm kiagent shows in the tools picker.
.vscode/mcp.json
{
  "servers": {
    "kiagent": {
      "url": "http://127.0.0.1:7421/mcp",
      "headers": {
        "Authorization": "Bearer ••••••••"
      }
    }
  }
}
Workspace-scoped: this snippet only takes effect in projects where you commit (or copy) the .vscode/mcp.json. For a global install, use User Settings → MCP Servers.
